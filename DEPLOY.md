✧ لحظة — نشر الباك اند على Render (بدون Ollama) ✧

================================
١) جيب مفتاح Gemini مجاني
================================
1. روح لـ https://aistudio.google.com/app/apikey
2. سجل دخول بحساب Google، دوس "Create API key".
3. انسخ المفتاح، هتحتاجه في الخطوة الجاية.

================================
٢) ارفع فولدر lahza-backend على GitHub
================================
1. اعمل repo جديد على GitHub (خاص أو عام، مش فارق — الفولدر ده مفيهوش أي سر).
2. ارفعله الملفات التلاتة: server.js, package.json, .env.example (اختياري).

================================
٣) اعمل Web Service على Render
================================
1. روح https://dashboard.render.com → New → Web Service.
2. اربطه بالـ repo اللي عملته.
3. الإعدادات:
   - Environment: Node
   - Build Command: npm install
   - Start Command: npm start
4. من تبويب Environment، ضيف المتغيرات دي:
   - GEMINI_API_KEY = المفتاح اللي جبته من خطوة ١
   - AGENT_KEYS = أي كلمة سر تختارها (مثلاً lahza-secret-123)
   - (اختياري) GEMINI_MODEL = gemini-2.0-flash
5. دوس Create Web Service واستنى الـ deploy يخلص (بياخد دقيقة أو اتنين).
6. هيديك رابط شكله كده:
   https://lahza-backend.onrender.com

ملحوظة: خطة Render المجانية بتـ"ينام" السيرفر لو مفيش طلبات لمدة، وأول طلب بعد النوم بياخد كذا ثانية لحد ما يصحى. لو عايزه يفضل صاحي دايمًا بسرعة، هتحتاج خطة مدفوعة بسيطة (Starter).

================================
٤) اربط الفرونت إند (اللي على فيرسل) بالباك اند الجديد
================================
افتح lahza-web.html ودوّر على السطرين دول في أول <script>:

    const API_BASE = "http://YOUR_SERVER_IP:8080";
    const AGENT_KEY = "REPLACE_WITH_CUSTOMER_1_KEY";

وغيّرهم لـ:

    const API_BASE = "https://lahza-backend.onrender.com";
    const AGENT_KEY = "lahza-secret-123";   // نفس القيمة اللي حطيتها في AGENT_KEYS على Render

بعد كده push للتعديل على الـ repo بتاع فيرسل، وهو هيعمل redeploy تلقائي.

================================
٥) تجربة
================================
افتح رابط فيرسل بتاعك واكتب أي رسالة. لو ظهرلك "لحظة متصلة وجاهزة" فوق، معناه كل حاجة اتظبطت صح.

================================
ملحوظات
================================
- الـ Gemini free tier بيدّيك عدد طلبات يومي محدود (بيتغير بين الحين والتاني حسب سياسة جوجل) — كفاية لتجربة واستخدام شخصي، لو حبيت تستخدمه تجاريًا بعدد عملاء كبير راجع أسعار وحدود https://ai.google.dev/pricing.
- لو عايز تبدّل لموديل تاني من Gemini (أسرع أو أدق)، غيّر قيمة GEMINI_MODEL بس في Render، مش محتاج تلمس الكود.
- توليد الصور والملفات المرفقة (Vision) لسه مش مفعّلين في النسخة دي، زي ما اتفقنا قبل كده.
